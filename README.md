### Persiapan
- PHP >= 8.2
- composer
- node & npm


### Instalasi
- buka terminal lalu masuk ke folder project
- jalankan perintah-perintah berikut
- composer install
- cp .env.example .env
- php artisan key:generate
- php artisan optimize:clear
- npm install
- npm run build
- php artisan serve
- setelah itu, buka alamat http://loclahost:8000 di browser
